DHTMLframe Dynamic HTML или DHTML — это способ (подход) создания интерактивного веб-сайта, использующий сочетание статичного языка разметки HTML, встраиваемого (и выполняемого на стороне клиента) скриптового языка JavaScript, CSS (каскадных таблиц стилей) и DOM (объектной модели документа). Динамический HTML понятие собирательное и представляет собой комбинацию некоторых новых тэгов HTML и возможностей программирования, что позволяет сделать веб-страницы интерактивными и размещать на них анимационные объекты. Под активностью тэгов динамического HTML понимается: 1) возможность идентификации каждого тэга или целого блока сгруппированных тэгов и доступа к идентифицированным тэгам и блокам тэгов из программ; 2) возможность доступа из программ к свойствам, методам и событиям каждого тэга. Идентификацию тэга можно осуществлять с помощью атрибутов ID и NAME. Кроме этого для доступа к стилям листов стилей используется атрибут CLASS. Для доступа из программ к объектам-тэгам в рамках объектной модели для этой цели существует общая стандартная коллекция всех элементов страницы, имеющая название all. Еще одна важная возможность, отсутсвующая в обычном HTML, но предусмотренная в динамическом HTML – это работа с нестандартными (неизвестными языку HTML) тэгами или, как их ещё называют – нераспознаваемыми элементами.
