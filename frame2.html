Управление окнами В динамическом HTML различают три типа окон немодальные окна; модальные окна; наборы фреймов. Немодальные окна (или просто «окна»), это окно экземпляра браузера, в котором открываются HTML-документы. Модальными называются окна, которые требуют ответа от пользователя для возобновления взаимодействия с браузером. Наборы фреймов рассматривались в подразделе 2.7. В конце настоящего раздела рассматриваются возможности программирования фреймов с использованием языка JavaScript. В данном подразделе показано, что динамический HTML имеет достаточный набор средств для манипулирования окнами путём программирования. Предусмотрены методы для создания, управления и осуществления перехода между тремя типами окон. Все эти методы предоставляются объектом windows в рамках объектной модели документа. В таблице 4.1 перечислены методы объкта windows, предназначенные для выполнения операций перемещения, изменения размера и прокручивания немодальных окон и фреймов. Все три операции могут быть выполнены относительно текущего состояния окна или нового абсолютного положения посредством пары методов для каждой операции.
