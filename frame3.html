DOM и активность HTML-тэгов Коллекции элементов, доступ к их свойствам, методам и событиям. Под активностью тэгов динамического HTML понимается: возможность идентификации каждого тэга или целого блока сгруппированных тэгов и доступа к идентифицированным тэгам и блокам тэгов из программ; возможность доступа из программ к свойствам, методам и событиям каждого тэга. Идентификацию тэга, как вы уже знаете, можно осуществлять с помощью атрибутов ID и NAME. Кроме этого, подразделе 4.3 вы узнаете, как для доступа к стилям листов стилей используется известный вам атрибут CLASS. Для доступа из программ к объектам-тэгам (далее мы будем их называть также элементами) в рамках объектной модели для этой цели существует общая стандартная коллекция всех элементов страницы, имеющая название all. Кроме этого, все элементы сгруппированы в стандартные коллекции по принадлежности, которые иногда более удобно использовать, чем общую коллекцию. Принадлежность элементов к тэгам легко определить из их наименования. Для браузера Internet Explorer в языке JavaScript (Jscript) предусмотрены следующие коллекции элементов: all, anchors, applets, areas, bookmarks, boundElements, cells, children, elements, embeds, filters, forms, frames, images, imports, links, mimeTypes, options, plugins, rows, rules, scripts, styleSheets, tBodies, TextRectangle. Эти коллекции являются массивами для программы и одновременно соответствуют тэгам HTML-документа. Коллекции являются объектами и одновременно свойствами объекта document. Поэтому на коллекцию all, например, можно ссылаться как на метод – document.all(). Манипулирование любой коллекцией осуществляется посредством набора свойств и методов, принадлежащих коллекциям. Первой наиболее полезной операцией с коллекциями является определение числа элементов в коллекции. Число элементов в коллекции возвращается свойством length. Доступ к элементам в коллекциях можно осуществлять несколькими способами. Для доступа можно использовать метод item. В качестве параметра для этого метода указывается либо порядковое положение, либо атрибуты name или id элемента. Далее, при использовании для идентификации элемента атрибутов name или id, будем называть такие элементы именованными элементами. Другой способ состоит в том, что к элементу в коллекции можно обращаться как к элементу массива, указывая в качестве индекса порядковый номер, либо атрибуты name или id элемента.
